Facundo's Games and Tools

About this site

I created this website to showcase my games and tools built over my time practicing new programming concepts. None of the games will work on a mobile website. Both tools, however, will.

All the games on this website are coded in client-side javascript using a library called p5js. The scrabble word value is written in vanilla client-side javascript. The food locator tool is also written in vanilla javascript, using the google maps places library.

This website itself is hand-written using Twitter's Bootstrap and Flexbox.
